При запуске программы создаёться 10 аккаунтов клиентов банка с id от 0 до 9, 
пароль у всех по умолчанию "0000":
1) Счёт(Id=0, psw = 0000), 2) Счёт (Id = 1, psw = 0000), и тд...
---------------------------------------------------------------------------
Так же существует акаунт работника банка с которого можно создавать новые счета
и удалять существующие. 
Id работника банка = 9999;
Пароль работника банка = "root";
---------------------------------------------------------------------------
С счетов можно переводить средства на другие счета, получать средства, менять пароль,
реализована история поступлений и отправлений.
---------------------------------------------------------------------------
P/S: Проэкт создавался для университета, по этому он на польском языке.
